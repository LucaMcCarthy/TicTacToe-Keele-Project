# TicTacToe-Keele-Project
Java TicTacToe game with console and GUI functionality. Completed as a project in Keele University during the second term. 

## Overview
This is a Java-based Tic-Tac-Toe game developed as a second-term project at Keele University.
The project includes a GUI and a console version, demonstrating object-oriented programming, event driven design and software testing skills via JUnit. 

For a full explanation of the project, code structure and included screenshots, see the [Project Report](report/TicTacToeReport.pdf)

## Features
- Interactive GUI with color-coded win/draw indicators
- Reset button functionality
- Console version for terminal play
- JUnit tests for verifying game logic

## Skills Demonstrated
Java, OOP, GUI Design, Event-Driven Programming, Software Testing, JUnit

## How To Play
1. Clone the repository
2. Open in your Java IDE
3. Run Main.java
   * This is the entry point for the project.
   * You will be prompted to choose which version:
     * Enter 1 for GUI
     * Enter 2 for Console
4. Enjoy Tic-Tac-Toe!

## Requirements
JUnit 4.13.2 - only if you want to run TicTacToeTest.java
